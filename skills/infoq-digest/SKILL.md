# infoq-digest - InfoQ 资讯抓取技能

🦐 抓取 InfoQ AI&大模型频道最新资讯，输出格式化列表

---

## 功能

1. 抓取 InfoQ AI&大模型频道最新资讯
2. 支持自定义条数（默认 15 条）
3. 输出格式化列表（标题 + 链接）
4. 支持飞书消息推送

---

## 使用方式

### 基础用法
```
抓取 InfoQ 最新 AI 资讯
```

### 自定义条数
```
抓取 10 条 InfoQ AI 资讯
```

### 推送飞书
```
抓取 InfoQ 资讯并推送到飞书
```

### Cron 定时任务配置
```yaml
sessionTarget: "main"
payload:
  kind: "systemEvent"
  text: "执行 infoq-digest 技能，抓取 InfoQ AI&大模型频道最新 15 条资讯并推送到飞书"
```

---

## 参数配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `count` | 15 | 抓取条数（1-50） |
| `url` | https://www.infoq.cn/topic/AI&LLM | 目标页面 |
| `format` | simple | 输出格式（simple/detailed） |
| `push` | false | 是否直接推送飞书 |

---

## 输出格式（标准版）

每条资讯必须包含以下字段：

```
### 1. 标题
**描述：** 摘要内容...
**作者：** xxx | **译者：** xxx（如有） | **时间：** 53 分钟前 | **热度：** 465
[🔗 原文链接](https://...)

---
```

### 字段说明

| 字段 | 必填 | 说明 |
|------|------|------|
| 标题 | ✅ | 完整标题 |
| 描述 | ✅ | 标题下方的摘要/简介 |
| 作者 | ✅ | 文章作者 |
| 译者 | ⚠️ | 如有译者则显示，无则省略 |
| 时间 | ✅ | 相对时间（如"53 分钟前"、"13 小时前"） |
| 热度 | ✅ | 阅读量/点赞数（如 465） |
| 链接 | ✅ | 文章 URL，Markdown 格式 |

### 排序规则
- 按时间倒序排列（最新的在前）

### 数量
- 默认抓取最新 15 条

---

## 实施步骤

1. **浏览器打开** InfoQ AI&大模型频道页面
2. **Snapshot 抓取** 页面列表内容（aria refs）
3. **解析提取** 标题、链接、作者（可选）
4. **格式化输出** 按指定格式整理
5. **推送飞书**（如配置 push=true）

---
