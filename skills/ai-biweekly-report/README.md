# AI 工程半月报技能 (ai-biweekly-report)

自动搜集、筛选、分析 AI 领域关键资讯，生成面向后端工程师转型 AI Engineer 的深度分析半月报。

## 文件结构

```
skills/ai-biweekly-report/
├── SKILL.md                  # 技能执行指令（agent 读取）
├── README.md                 # 本文件
└── references/
    └── template.md           # 输出格式示例
```

## 使用方式

### 手动触发

```
请生成 2026 年 4 月上半月 AI 工程半月报（4 月 1 日 -15 日），folder_token: xxxxx
```

也可通过配置定时任务 Cron 实现自动化触发。

## 执行流程概览

```
确认参数（时间范围 + folder_token）
  → 多数据源搜索（核心 5 + 备选 3）
  → 两阶段日期过滤（摘要初筛 + web_fetch 精确核实）
  → 筛选 8-12 条 + 按主题聚合
  → 提炼 2-3 条跨资讯趋势洞察（含行动建议）
  → 资讯速览（全部，含上手门槛标签）+ 重点深度分析（Top 3-5）
  → 创建飞书文档 + 发送摘要
```

## 前置要求

- **飞书 folder_token**：必须由用户提供，用于指定文档创建的目标文件夹

## 配置参数

| 参数 | 可选值 | 说明 |
|------|--------|------|
| 时间范围 | 上半月 / 下半月 / 自定义 | 报告覆盖的时间段 |
| 交付格式 | 飞书文档 / Markdown / 两者 | 输出形式 |
| 资讯比例 | 产业 4-6 / 学术 2-4 / 实践 1-3 | 弹性分配，总计 8-12 条 |
