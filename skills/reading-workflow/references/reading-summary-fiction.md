# 小说 / 虚构 — 读后总结段模板

> 本模板是「📖 读后总结」段落的子结构，由 `+summary` 注入到 `reading/{key}.md` 里。
> 不要重复进度表、章节笔记、感想等主文件已有的内容，只放类型独有的回看提炼。

---

### 故事核心
- 一句话概括
- 核心人物关系（表格：人物 / 关系 / 关键事件）

### 主题与象征
- 表格：主题 / 体现方式 / 个人解读

### 人物分析
- 主要人物深度分析（表格：人物 / 性格 / 弧光 / 关键选择）

### 叙事手法
- 表格：手法 / 场景示例 / 效果

### 核心洞察与感悟
- 从过程感想中提炼的、读完才看清的东西
- 与读前期待的对比（呼应「🚀 阅读计划」里的想法 / 期待）

### 未解之谜
- 作者故意留白的问题（表格：问题 / 我的猜测）

### 总体评价
- 多维度评分（表格：维度 / 评分 / 理由）

### 这本书给我的启发
- 1-3 条具体的、可带出书本之外的东西
