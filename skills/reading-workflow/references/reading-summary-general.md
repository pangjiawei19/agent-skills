# 通用 / 非虚构 — 读后总结段模板

> 本模板是「📖 读后总结」段落的子结构，由 `+summary` 注入到 `reading/{key}.md` 里。
> 不要重复进度表、章节笔记、感想等主文件已有的内容，只放类型独有的回看提炼。

---

### 核心论点
- 一句话概括作者的主要观点
- 论证方式（案例 / 数据 / 推演 / 对话）

### 关键框架与模型
- 书中提出或引用的框架（表格：框架 / 构成 / 适用场景）

### 关键论据
- 重要案例 / 数据 / 研究（表格：论点 / 论据 / 可信度）

### 核心洞察与感悟
- 触动我的观点
- 与我已有认知的印证或冲突

### 如何应用
- 能迁移到哪些场景
- 具体可落地的做法

### 疑问与批判
- 存疑的论点
- 作者没讨论的反例或局限

### 总体评价
- 多维度评分（观点原创性 / 论证严谨性 / 可读性 / 启发性）

### 这本书给我的启发
- 1-3 条具体的、可带出书本之外的东西
